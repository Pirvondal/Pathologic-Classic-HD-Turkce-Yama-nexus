# Pathologic Classic HD Türkçe Yama – Katkı Rehberi

## Proje Hakkında

Bu proje, Pathologic Classic HD oyununu Türkçeye çevirmek amacıyla hazırlanmıştır.

Toplamda 28.000+ satır metin çevrilmiştir. Ancak hâlâ:

* Anlam hataları
* Anlatım bozuklukları
* Tutarsız çeviriler

bulunabilir. Bu yüzden katkılar çok değerlidir.

## Nasıl Katkı Sağlayabilirim?

### 1. Çeviri Düzelt

En basit ve en önemli katkı:

* Daha doğal Türkçe yaz
* Anlamı düzelt
* Garip cümleleri iyileştir

### 2. Hata Bildir (Kolay Yol)

Kod bilmeden de katkı yapabilirsin:

1. Issues sekmesine gir
2. Yeni konu aç
3. Şunu yaz:

```
Orijinal metin:
...

Mevcut çeviri:
...

Önerilen çeviri:
...
```

### 3. Doğrudan Düzenle (İleri Seviye)

Biliyorsan:

1. Repoyu forkla
2. Dosyayı düzenle
3. Pull Request gönder

## Nelere Dikkat Edilmeli?

* Anlamı değiştirme
* Oyunun karanlık ve edebi havasını koru
* Aynı kelimeleri tutarlı çevir
* Aşırı gündelik veya slang kullanma

## Neler Çevrilemez?

* Menü butonları vb. görsel (texture) olduğu için değiştirilemez

Katkıda bulunan herkese teşekkürler.
Bu proje birlikte daha iyi olacak.
